Custom Icon with the possibility to display an informative text when the user interact with this component

Since

1.1.5

Parameters

modifier

The Modifier to be applied to this icon

infoTextColor

The color to apply to the infoText

infoTextBackgroundColor

The color to apply to the background of the infoText

size

The size of the icon

imageVector

The ImageVector to draw inside this icon

infoText

The informative text about what the icon represents, it is used also as contentDescription

tint

The tint to be applied to imageVector. If Color.Unspecified is provided, then no tint is applied
