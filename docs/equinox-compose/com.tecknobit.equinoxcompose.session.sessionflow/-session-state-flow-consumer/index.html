SessionStateFlowConsumer

interface SessionStateFlowConsumer

The SessionStateFlowConsumer interface allows consuming a SessionFlowState instance and provides methods to handle its lifecycle with dedicated wrapper methods.

This, for example, it is useful to apply to an com.tecknobit.equinoxcompose.viewmodels.EquinoxViewModel to directly invoke the methods that handle the session state using the provided lifecycle methods
