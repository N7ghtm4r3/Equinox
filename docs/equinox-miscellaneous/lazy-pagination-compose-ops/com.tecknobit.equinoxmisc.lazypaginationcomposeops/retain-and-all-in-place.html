retainAndAllInPlace

@ExperimentalPaginationApi
fun <KEY, T> PaginationState<KEY, T>.retainAndAllInPlace(items: Collection<T>?, nextPageKey: KEY, isLastPage: Boolean = false)

Method split in two phases used to first remove all the items from the current PaginationState.allItems that are not present in the items collection, then add all the items present in the same items collection but not in the main one. For example:

val paginationState = PaginationState<Int, Int>(
    initialPageKey = 0,
    onRequestPage = { page ->
        appendPage(
           nextPageKey = page + 1,
           items = listOf(1, 2, 3)
        )
    }
)

paginationState.retainAndAllInPlace(
    items = listOf(2, 3, 4),
    nextPageKey = 2,
    isLastPage = true
)

// the paginationState will have: 2, 3, 4
