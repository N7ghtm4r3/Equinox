# Debounced inputs

These components allow to you to implement a debounce routine handled when the user stop to typing in the inputs fields

## Usage

### DebouncedTextField

```kotlin
@Composable
fun App() {
    DebouncedTextField(
        value = state,
        placeholder = "your_placeholder",
        debounceDelay = ,// custom delay default 500 ms
        debounce = {
            // your debounce logic here
        }
    )
}
```

### DebouncedOutlinedTextField

```kotlin
@Composable
fun App() {
    DebouncedOutlinedTextField(
        value = state,
        placeholder = "your_placeholder",
        debounceDelay = ,// custom delay default 500 ms
        debounce = {
            // your debounce logic here
        }
    )
}
```
