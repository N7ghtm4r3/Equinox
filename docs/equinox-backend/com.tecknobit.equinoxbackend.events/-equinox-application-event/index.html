EquinoxApplicationEvent

abstract class EquinoxApplicationEvent<T : Enum<out Any>?> : ApplicationEvent

The EquinoxApplicationEvent is the class used to share the information of an event emitted by the EquinoxEventsEmitter allowing the dedicated EquinoxEventsCollector to retrieve and use that information. It is based on top of the ApplicationEvent API
